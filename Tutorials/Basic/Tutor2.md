# Возможности для работы со строками
## Добавка/сложение строк
Так как для вывода нескольких строк писать постоянно по два, а то и больше раза `Console.WriteLine()` сомнительное удовольствие, вы можете просто складывать строки.<br>
Итогом вы сможете за один `Console.WriteLine()` вывести сразу всё что вам нужно будет. А вот и пример:<br>
```csharp
Console.WriteLine("Антон" + "Чигур" + "не" + "убивал");
//выведет "АнтонЧигурнеубивал"
```
*просто отмечу, что принято между элементами и плюсами делать пробелы, не смотря на то, что это не обязательно*

Хорошо, мы можем уже выводить много текста, но он слитный, вопрос как сделать пробелы уже есть у тех, кто не особо разбирается в области информатики.<br>
На самом деле всё досмеху просто, пробел такой же символ, как и остальные буквы, просто пустой. 
Итогом, нам достаточно просто написать этот пробел, и у нас уже будут разделения. Смотрите сами:
```csharp
//добавляем пробелы чтобы текст был раздельным
Console.WriteLine("Антон " + "Чигур" + " не " + "убивал");
//выведет "Антон Чигур не убивал", нужный результат достигнут
```
## Функции управления последовательности
Звучит заумно, но на самом деле всё просто.<br>
Данные функции представляют из себя обратный слэш (`\`) с нужной буквой и/или значением
Для удобства, я составил таблицу с большенством таких функций, и мелким примером кода для них

**Функции управления последовательностью**:
1. `\n` - снос строки. Переносит курсор на строку вниз. 
```csharp
Console.Write("Этот текст\nстранный);
/*Выведет:
"Этот текст
странный"
*/
```
2. `\t` - горизонтальная табуляция. Создаёт отступ от начала строки. 
```csharp
Console.Write("\tМатематическое пи = 3,14");
//Выведет: "  Математическое пи = 3,14"
```
3. `\r` - возврат курсора. Переносит курсор в начало строки (Пример не нужен, я полагаю)
4. `\\` - Используется чтобы написать сам символ `\` (Иначе то что после, будет восприниматься компилятором как функция)
5. `\b` - бэкскейп. Удаление последнего напечатанного символа.
```csharp
Console.WriteLine("ВВП\b");
//Выведет "ВВ", т.к. "П" была удалена
```
6. `\u + код` - позволяет ввести символ юникода, используя его 16-тиричный код (Например `\u00A9` выведет знак copyright)
Однако всё это не удобно вставлять в больше текста, тем более, после добавления всего этого текст для программиста превращается в кашу.<br>
Именно по этому придумали более удобный способ написания подобных текстов, и недавно его добавили.
## Verbatim-строки
Сама суть Verbatim-строк, убрать из строк функции управление последовательностью, заменив их физическим изменением в тексте.<br>
Смотрим пример:
```csharp
//Для использования Verbatim, надо перед `"` добавить `@`
Console.Write(@"В этом тексте
Сохранится этот перенос строки
  И этот отступ");
//Вывод будет содержать весь тот текст что мы написали со всеми манипуляциями в нём
```
Однако сам минус Verbatim-строк, это невозможность использовать другие "`\` функции", например, мы не сможем использовать в тексте из примера `\b`.<br>

Всё это хорошо, но это позволяет нам только выводить что-то в консоль... НАМ. А вдруг пользователь тоже что-то хочет?
## Ввод в программу от пользователя
Т.к. программы работают на приёме и отдаче данных, а мы знаем только про отдачу (`Console.WriteLine()`) данных, пора узнать про приём их.<br>
Для это используются методы `Console.ReadLine()`, они создают поле, в которое можно ввести текст, символ, или которые принимают ввод. Вот различия:
1. `Console.ReadLine()` - Используется для приёма длинных (Ну как, 2+ буквы) текстов, после чего продолжает программу.
2. `Console.Read()` - Используется для приёма одного символа, или просто продолжения программы по нажатию Enter.
3. `Console.ReadKey()` - Принимает любое нажатие кнопки от пользователя, и затем продолжает программу.
Используем в примере метод 1:
```csharp
Console.WriteLine(Console.ReadLine());
//Пользователь введёт что-то, и этот же текст выведется в консоль
```
